# TruEstate Backend API

Backend service for the TruEstate Sales Management System.

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB with Mongoose ODM
- **Environment**: dotenv for configuration

## Features

- RESTful API for sales data management
- Advanced filtering with multi-select support
- Full-text search on customer name and phone
- Flexible sorting options
- Efficient pagination
- Database indexing for performance

### Prerequisites
